User Requirement Specification Document

Dokumen URSD berisi rangkuman kebutuhan sistem, mencakup kebutuhan fungsional dan non-fungsional, prioritas kebutuhan, serta kriteria keberhasilan untuk menilai tercapainya tujuan aplikasi. Di dalamnya juga terdapat model tugas untuk menggambarkan aktivitas pengguna, use case diagram sebagai alur interaksi sistem, dan site map aplikasi yang menunjukkan struktur navigasi secara keseluruhan.
